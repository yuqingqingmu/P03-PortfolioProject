Web - E-Commerce Website

About

This is a project for creating a e-commerce website for a visual interior design company. As a team, we applied a minimalism style to design the website in order to fit the concept of the company. This website contained all the basic information that a company should have, including the update event, company story, product list and contact method. As the designer, we would like to create a clear simple layout website with accessibility in order to let all the users can browser the website easily.

My Role

Home Page Designer & Developer

Detail Page Designer & Developer

APPLIED SKILLS/TOOLS

GitKarden, HTML, CSS, Javascript

Process

Step 1 - Styleguide

Basing on the atomic design priciple, we have created a styleguide in order to decide all the elements used in the website, including text input, buttons and so on. The color pallete and style were also shown in this stage. Click to view the style guide.

Step 2 - Wireframe

Starting to create the wireframes of detail page by Wix. In this page, I would like to show what's kind of service that the company could offer. By using different font size to seperate the service's name with the introduction of it. Besides, the recommand product would be also displayed on this page.

Step 3 - Html & Css

Developing the website with the real content and images by applying Html and Css code. Adding more animation effects on the interactive elements, for instance, buttons and navigation bar.

Step 4 - Refine responsivness

Using the Inspect Elements from Chrome, and fixing the responsive problem. In this stage, I have applied the flex box in order to make different section transfer to different layout basing on the screen size. For instance, the display of the introduction of the service would become vertical on the mobile.

Step 5 - Usability Testing

This is the final stage of the process. In order to improve the website, we have recieved lots of feedback from peers. Basing on their feedback, I did following changes:

  • Adding the width of the update event section to make more words in single line.
  • Making the padding of the content smaller.
  • Splitting different section with a border.
  • Making the font size smaller in order to suite the small screen.

Challenge & Solution

The biggest challenge is to design a user-friendly website for all the users. At the first, the layout of web page is not responsive on different devices, for instance, all the text were compressed together on iphone; the image is too small for some; and the navigation bar can not work, even they were fine on the computer.

Solution

In order to solve the problem, we conducted usability test on different devices and fix each problem. Adding the flex box for div could be really helpful for creating a responsive web page.

Reflection

After this project, I have learned more knowledge about website designing. First, creating a styleguide at the beginning could help a team to make sure the style and direction, and the usability and accessibility are important for a website, since we have different users are using different devices.
